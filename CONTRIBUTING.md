**Panduan Kontribusi**

Terima kasih telah berminat berkontribusi ke proyek ini.

Cara berkontribusi singkat:

1. Fork repository ini ke akun GitHub Anda.
2. Buat branch fitur atau perbaikan: `git checkout -b feature/nama-anda`.
3. Buat perubahan yang jelas dan kecil; sertakan penjelasan pada commit.
4. Pastikan tidak menyertakan file Excel berisi data sensitif.
5. Push branch Anda dan ajukan Pull Request ke repository utama.

Catatan:
- Untuk perubahan kode VBA, sertakan penjelasan singkat pada PR dan contoh langkah reproduksi jika perlu.
- Jika PR mengubah struktur sheet/kolom, perbarui `docs/STRUKTUR-DATABASE.md`.

Terima kasih!
